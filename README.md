
# City Bank Global

This is the official platform for City Bank Global (branding: Nexa International).

## Features
- **User Registration:** Secure institutional profile provisioning.
- **Asset Management:** Unified dashboard for checking, savings, and corporate accounts.
- **Transaction Ledger:** Real-time, chronologically ordered financial auditing.
- **Capital Movement:** Secure domestic and cross-border settlements.
- **AI Support:** Specialized assistant for institutional inquiries.

## Technologies Used
- Next.js 15 (App Router)
- Tailwind CSS & Shadcn UI
- Firebase Authentication & Firestore
- Genkit (Gemini 2.5 Flash)
